# GramaKhata 📒

GramaKhata is a simple Android ledger application built for local shops and small businesses to manage customer credit transactions digitally.

## Features

* Add/Edit/Delete Customers
* GIVE & TAKE Transaction System
* Automatic Balance Calculation
* Transaction History
* WhatsApp Payment Reminder
* SMS Reminder Support
* Customers Sorted by Highest Due
* SQLite Offline Database
* Search Customers

## Technologies Used

* Kotlin
* Android Studio
* SQLite Database
* RecyclerView
* Material Design Components

## Future Enhancements

* PDF Report Export

* Multi-language Support
* Analytics Dashboard
